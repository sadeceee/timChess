# TimChess
Inspired by FBchess wit Forsyth-Edwards-Notation.

  Instructions: 
    run with 'java -jar PATH_TO_JAR_FILE.jar' in terminal
    
    play: Type input and press go
      input: [previous square][target square]
      example input: e2e4
                     e7e5
      castling input: 
        short castling: o-o
        long castling: o-o-o
